# Calculator Project

A simple command-line calculator that performs basic arithmetic operations.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Exponentiation (^)

## Usage

```bash
python calculator.py
```

## Example

```
Enter expression (e.g., 5 + 3): 10 * 5
Result: 50.0
```

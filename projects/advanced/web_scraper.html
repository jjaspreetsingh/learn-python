Project: Web Scraper

The web is full of data. Learn how to write a script that visits websites and extracts the information you need automatically.

1. The Tools

We'll use two popular libraries:

  • requests: To download the HTML of a page.
  • BeautifulSoup: To find specific data inside that HTML.

2. The Code

import requests
from bs4 import BeautifulSoup
import time

def get_quotes():
    # Target Website (Safe for scraping tests)
    url = "http://quotes.toscrape.com"
    
    # 1. Get the page
    response = requests.get(url)
    
    # 2. Parse HTML
    soup = BeautifulSoup(response.text, "html.parser")
    
    # 3. Find elements
    quotes = soup.find_all("div", class_="quote")
    
    for quote in quotes:
        text = quote.find("span", class_="text").text
        author = quote.find("small", class_="author").text
        print(f"\"{text}\" - {author}")

        # Be nice to the server!
        time.sleep(0.5)

if __name__ == "__main__":
    get_quotes()

⚠️ Ethics Check: Always check a website's robots.txt file before scraping. Don't hit servers too fast; use time.sleep() to be polite.

Project: Build an API

Websites talk to servers using APIs (Application Programming Interfaces). Let's build a simple one using Flask.

1. Setup

pip install flask

2. The Code

This simple app lets you get a list of items and add new ones.

from flask import Flask, jsonify, request

app = Flask(__name__)

# Simple in-memory database
items = [{"id": 1, "name": "Potion"}, {"id": 2, "name": "Sword"}]

@app.route('/items', methods=['GET'])
def get_items():
    return jsonify(items)

@app.route('/items', methods=['POST'])
def add_item():
    new_item = request.get_json()
    items.append(new_item)
    return jsonify(new_item), 201

if __name__ == '__main__':
    app.run(debug=True)

3. Testing

You can test your API using a browser (for GET requests) or tools like Postman or `curl`.

# Get items
curl http://127.0.0.1:5000/items

# Add item
curl -X POST -H "Content-Type: application/json" -d '{"id": 3, "name": "Shield"}' http://127.0.0.1:5000/items

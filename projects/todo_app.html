Project: Todo List App

Every programmer builds a Todo App. It's the "Hello World" of real applications. You'll learn to manage a list of data.

1. The Plan

  1. Create an empty list to store tasks.
  2. Create a loop that asks the user what they want to do (Add, View, or Quit).
  3. Write functions for each action.

2. The Code

tasks = []

def show_tasks():
    if not tasks:
        print("Your list is empty.")
    else:
        print("\nTo-Do List:")
        for i, task in enumerate(tasks, 1):
            print(f"{i}. {task}")
    print()

while True:
    print("1. Add Task")
    print("2. View Tasks")
    print("3. Quit")
    
    choice = input("Enter choice: ")
    
    if choice == '1':
        item = input("Enter task: ")
        tasks.append(item)
        print("Task added!")
    elif choice == '2':
        show_tasks()
    elif choice == '3':
        print("Goodbye!")
        break
    else:
        print("Invalid choice.")

Pro Tip: enumerate(tasks, 1) gives you both the index (counting from 1) and the item, which is perfect for numbered lists.

3. Challenge

Add a "Delete Task" feature! You'll need to ask the user which number to remove and use tasks.pop().

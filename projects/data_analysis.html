Project: Simple Data Analysis

Python is the #1 language for Data Science. Let's do some basic analysis without any fancy libraries first.

1. The Dataset

Imagine we have a list of student scores.

scores = [88, 92, 79, 93, 85, 91, 76, 95]

2. Analysis Functions

Let's calculate the Mean (Average) and find the Max score.

def calculate_average(data):
    total = sum(data)
    count = len(data)
    return total / count

average = calculate_average(scores)
highest = max(scores)
lowest = min(scores)

print(f"Average Score: {average}")
print(f"Highest Score: {highest}")
print(f"Lowest Score: {lowest}")

3. Using Standard Libraries

Python has a built-in statistics module.

import statistics

print(f"Median: {statistics.median(scores)}")
print(f"StDev: {statistics.stdev(scores)}")

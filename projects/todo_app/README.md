# Todo App Project

A simple command-line todo list application with persistent storage.

## Features

- Add new todos
- List all todos
- Mark todos as completed
- Delete todos
- Persistent storage using JSON

## Usage

```bash
python todo.py
```

## Commands

- `add` - Add a new todo
- `list` - Display all todos
- `complete` - Mark a todo as completed
- `delete` - Delete a todo
- `quit` - Exit the application

## Data Storage

Todos are saved in `todos.json` file in the same directory.

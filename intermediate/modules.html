Modules & Libraries

You don't have to write everything yourself. Python has a massive ecosystem of Modules (libraries) that you can import and use.

1. Importing Modules

Python comes with "batteries included". You can use built-in modules like math, random, or datetime immediately. 

import math
import random

print(math.sqrt(16))      # Output: 4.0
print(random.randint(1, 10)) # Output: Random number 1-10

2. Pip: The Package Installer

To use code from other people (like for web development or data science), you install it using pip in your terminal. 

pip install requests
pip install pandas

3. Using Installed Packages

import requests

response = requests.get("https://www.google.com")
print(response.status_code) # Output: 200 (OK)

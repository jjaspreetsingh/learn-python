Functions: Reusable Code

Imagine you have a recipe for a cake. You don't rewrite the recipe every time you want to bake it; you just follow the instructions. Functions are like recipes for your code.

1. Defining a Function

Use the def keyword to create a function. You give it a name and parameters (ingredients). 

def greet(name):
    print("Hello, " + name + "!")

# Calling the function
greet("Alice")
greet("Bob")

2. Return Values

Functions can give something back using the return keyword.

def square(number):
    return number * number

result = square(4)
print(result)  # Output: 16

Tip: Once a function hits return, it stops immediately. Any code after return won't run.

3. Scope

Variables created inside a function are "local"—they only exist inside that function.

def check_scope():
    secret = "I am hidden"

check_scope()
# print(secret)  # This would cause an Error!

Python Cheat Sheets

Quick references for when you forget the syntax.

Strings

s = "Hello World"
len(s)           # Length: 11
s.upper()        # "HELLO WORLD"
s.lower()        # "hello world"
s.replace("H", "J") # "Jello World"
s[0]             # "H" (First letter)
s[-1]            # "d" (Last letter)

Lists

lst = [1, 2, 3]
lst.append(4)    # [1, 2, 3, 4]
lst.pop()        # Removes last item
lst[0]           # First item
lst[1:3]         # Slice: [2, 3]

Dictionaries

d = {"name": "Alex", "age": 10}
d["name"]        # "Alex"
d.keys()         # ["name", "age"]
d.values()       # ["Alex", 10]

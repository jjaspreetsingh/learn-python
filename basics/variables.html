Variables & Data Types

Welcome to Python! The first thing you need to know is how to store information. Think of a Variable as a labeled box where you can put data to use later.

1. Creating a Variable

In Python, you don't need to do anything special to create a variable. You just give it a name and assign a value using the equals sign =.

# This puts the text "Alice" into a box labeled 'name'
name = "Alice"

# This puts the number 25 into a box labeled 'age'
age = 25

print(name)  # Output: Alice
print(age)   # Output: 25

2. Common Data Types

Python has a few main types of data you'll use all the time:

  • String (str): Text. Always surrounded by quotes ("Hello" or 'Hello').
  • Integer (int): Whole numbers (e.g., 5, 100, -42).
  • Float (float): Decimal numbers (e.g., 3.14, 9.99).
  • Boolean (bool): True or False values. Used for logic.
# String
city = "New York"

# Integer
population = 8000000

# Float
temp = 98.6

# Boolean
is_sunny = True

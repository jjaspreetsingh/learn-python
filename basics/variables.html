Variables & Data Types

Variables are containers for storing data values. In Python, you don't need to declare the type of a variable; Python automatically determines it based on the value assigned.

Creating Variables

Use the assignment operator (=) to create variables:

name = "Alice"
age = 25
height = 5.7
is_student = True

Data Types

Python has several built-in data types:

  • Strings (str): Text data, enclosed in quotes
  • Integers (int): Whole numbers
  • Floats (float): Decimal numbers
  • Booleans (bool): True or False values

Example:

# String
message = "Hello, World!"

# Integer
year = 2023

# Float
pi = 3.14159

# Boolean
is_raining = False

print(message)
print(year)
print(pi)
print(is_raining)

Type Conversion

You can convert between data types using built-in functions:

num_str = "42"
num_int = int(num_str)  # Converts string to integer

price = 19.99
price_int = int(price)  # Converts float to integer (truncates)

count = 5
count_str = str(count)  # Converts integer to string
Tip: Use type() function to check a variable's data type: print(type(variable_name))

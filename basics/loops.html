Loops: Repeat Actions

Programmers are lazy—in a good way! If you want to print "Hello" 50 times, you don't write it 50 times. You use a Loop.

1. The For Loop

Use a for loop when you want to iterate over a list of items or a specific range of numbers. 

# Loop through a list of fruits
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# Loop 5 times (0 to 4)
for i in range(5):
    print(i)

2. The While Loop

Use a while loop when you want to keep repeating as long as a condition is true. 

count = 0
while count < 5:
    print("Count is:", count)
    count = count + 1

⚠️ Infinite Loop Warning: If you forget to update the condition (like increasing count), the loop will run forever until your program crashes!

3. Break and Continue

  • break: Stops the loop immediately.
  • continue: Skips the rest of the current loop iteration and starts the next one.

Loops

Loops allow you to execute a block of code repeatedly. Python has two main types of loops: for loops and while loops.

For Loops

For loops iterate over a sequence (like a list, string, or range):

for item in sequence:
    # code to execute for each item

Example:

fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)

Range Function

Use range() to generate a sequence of numbers:

for i in range(5):  # 0, 1, 2, 3, 4
    print(i)

for i in range(1, 6):  # 1, 2, 3, 4, 5
    print(i)

While Loops

While loops continue as long as a condition is True:

while condition:
    # code to execute
    # update condition

Example:

count = 0

while count < 5:
    print(count)
    count += 1

Loop Control Statements

  • break - Exit the loop immediately
  • continue - Skip the rest of the current iteration

Example with break:

for i in range(10):
    if i == 5:
        break
    print(i)  # Prints 0, 1, 2, 3, 4

Example with continue:

for i in range(5):
    if i == 2:
        continue
    print(i)  # Prints 0, 1, 3, 4

Nested Loops

You can put loops inside loops:

Example:

for i in range(3):
    for j in range(2):
        print(f"i={i}, j={j}")

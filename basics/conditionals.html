Logic & Conditionals

Programs need to make decisions. "If it's raining, take an umbrella." In Python, we use if, elif, and else to control the flow of the program.

1. The If Statement

The if statement checks if a condition is True. If it is, the indented code block runs.

age = 20

if age >= 18:
    print("You are an adult.")
    
# Output: You are an adult.

⚠️ Important: Indentation matters! The line after the : must be indented (usually 4 spaces). This tells Python "this code belongs to the if statement".

2. Else and Elif

What if the condition isn't true? Use else to handle "everything else", or elif (else if) to check specific alternative conditions.

score = 85

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
else:
    print("Grade: C or lower")

3. Logical Operators

You can combine conditions using and, or, and not.

  • width > 10 and height > 10 (Both must be true)
  • day == "Sat" or day == "Sun" (Either can be true)
  • not is_hungry (Flips true to false)
